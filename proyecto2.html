Desktop App - Registro de profesores


WPF, .NET

Aplicación de escritorio destinada al registro de profesores, perteneciente a un trabajo grupal para la universidad, está escrita con C# la lógica de negocio y XAML para la interfaz. Tiene funcionalidades CRUD. Cuenta con una arquitectura MVVM, base de datos MySQL.
