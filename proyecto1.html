Librería Delta

API destinada a la gestión de libros con funcionalidades como: registrar, modificar, ver, eliminar y buscar tanto libros como autores y editoriales.
Si un usuario se registra tiene la posibilidad de convertirse en admin a través de un código específico y desbloquear funciones de creación y modificación.
Está conectado a una base de datos SQL e implementa Spring Security para el manejo de autenticación y autorización de usuarios, utilicé Docker para el empaquetado y Railway + Render para el despliegue. Su front-end está desarrollado con HTML5, CSS3, Bootstrap.
Éste es un proyecto personal que comenzó con la intención de aprender Spring Boot, pretende respetar los principios de la programación orientada a objetos y la confidencialidad de las variables de entorno.
